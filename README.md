# PTNR Advisory — Static Website

A pure HTML + CSS marketing site for PTNR Advisory.

## Pages
- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `impact.html` — Impact / case studies
- `contact.html` — Contact form

## Stack
- Plain HTML5
- Single shared `styles.css` (no build step)
- Inline SVG icons (no JS dependency)
- Google Fonts (Montserrat, Inter, JetBrains Mono) loaded via CDN
- Tiny inline script for the mobile menu toggle and contact form thank-you state

## Hosting on GitHub Pages
1. Push this folder to a GitHub repo (e.g. `ptnr-advisory`).
2. In **Settings → Pages**, set the source to the `main` branch, root (`/`).
3. Your site will be available at `https://<your-username>.github.io/<repo-name>/`.

No build step or framework required.
